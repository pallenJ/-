### 데이터 정렬

- ORDER BY

```sql
SELECT * FROM `table_name` ORDER BY `col`;
```
>알파벳, 날짜, 숫자 등 데이터 타입에 따라 순서대로 정렬
>기타 데이터 정렬은 DB 설정에  

- 위치로 정렬

```sql
SELECT * FROM `table_name` ORDER BY 2,3;
```
>칼럼의 상대적인 위치를 적음. 
>다시 컬럼명을 적지 않아도 되나 잘못된 칼럼을 적을 수 있으니 유의

- 정렬 순서
```sql
SELECT * FROM `table_name` ORDER BY `col` DESC;
```
> DESC는 역 정렬, ASC는 원래대로.
> 명시된 칼럼에만 적용
