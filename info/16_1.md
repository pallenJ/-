### PHP 코드 영역

PHP 코드는 웹 페이지의 어느 부분에나 위치할 수 있다.<br>
따라서 PHP 파서가 PHP 코드를 인식하기 위해서는 그 시작과 끝을 알려줄 필요가 있다.<br>
PHP 코드 영역을 알려주는 방식에는 다음과 같은 스타일들을 사용할 수 있다.

```php
 1. PHP 권장 스타일      : <?php ... ?>
 2. HTML 스크립트 스타일 : <script language = "php"> ... </script>
 3. SGML 스타일          : <? ... ?>
 4. ASP 스타일           : <% ... %>
```
